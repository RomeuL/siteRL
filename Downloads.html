VSCode
O Visual Studio Code (VSCode) é um ambiente de desenvolvimento integrado (IDE) altamente popular e amplamente utilizado, conhecido por sua eficiência, extensibilidade e interface amigável. Desenvolvido pela Microsoft, o VSCode é uma ferramenta poderosa para desenvolvedores de software em uma variedade de linguagens de programação.


Xcode
O Xcode é um ambiente de desenvolvimento integrado (IDE) desenvolvido pela Apple para a criação de aplicativos para seus dispositivos, como iPhone, iPad, Mac, Apple Watch e Apple TV. Ele é uma ferramenta poderosa e abrangente, projetada especificamente para desenvolvedores que trabalham com as tecnologias da Apple, como Swift, Objective-C e SwiftUI.


GitHub
O GitHub é uma plataforma de desenvolvimento de software baseada na web, amplamente utilizada por desenvolvedores e equipes de todo o mundo para hospedar, revisar e colaborar em projetos de código aberto e privados. Fundado em 2008, o GitHub oferece uma variedade de recursos e ferramentas que facilitam o processo de desenvolvimento de software em equipe. É muito bom pra roubar códigos também.
